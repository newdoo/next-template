## How to use

인스톨 및 실행:

```bash
npm run dev:web
```

이미지 파일
src/client/static

클라이언트
src/client

서버
src/server

사용된 라이브러리
next.js           서버사이드 react
material-ui       ui
crypto-promise    암호화 config.json 참조
isomorphic-fetch  back-end + eos 와 http 통신
mobx              redux 대체
express           back-end용 http web 서버 시작
truffle           이더리움 인터페이스
